Need help choosing the best HXL hashtags for your data? Find hashtag examples and definitions, and see how data managers are using the hashtags in their data. If you need help, send a tweet to @humdata with hashtag #HXL or send an email to hdx@un.org. Our team is here to help you. Visit HXL Standard to learn more.
HDX Tools, including Quick Charts, HXL Tag Assist and Data Check, aims to help humanitarians to clean and visualize data. The HDX Tools domain is not a secure service. Users should not use HDX Tools to process personal or otherwise sensitive data. Data uploaded to HDX Tools is not retained within the HDX infrastructure, while data downloaded by HDX Tools from public URLs is cached only as long as necessary for processing. Please reach out to hdx@un.org with any related questions or comments.
